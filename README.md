# Freddy Landing (20/80)

This workspace contains a continuous-scrolling landing page implementing a pseudo-3D mesh background, an interactive pipeline visualizing the 20/80 model, a full native contract in an interactive terminal box, and social connection nodes.

Quick start

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Notes
- The UI uses `framer-motion` for hover/tilt interactions.
- Customize contact links in `components/SocialNodes.jsx`.
- Contract text is in `components/TerminalContract.jsx` and is intentionally kept native and unabridged.
